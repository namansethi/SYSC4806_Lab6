Get your address books here

Listed below are the REST calls possible.

To get all address books = GET /addressbook

To get a specific address book (id is the address book's id) = GET /addressbook/{id}

To add a buddy to an address book (id is the address book's id) (BODY: String Name, Long Number) = POST /addressbook/{id}

To delete a buddy from an address book (id is the address book's id) (BODY: String Name, Long Number) = DELETE /addressbook/{id}
